Snapping and Docking

A smart desktop has form that facilitates function. All applications launched by Finsemble share an aesthetic which can be white labeled. Each window can participate in a range of elegant UX features. Windows launched by Finsemble are persistent between sessions, reloading in the same position and with the same content.
